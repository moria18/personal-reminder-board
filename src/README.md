# DIGITAL NOTICE BOARD
# Personal Reminder Board

## Description
A user-centered digital notice board for personal and academic reminders. 
This design allows you to add,view, and manage reminders with a soft, girly theme.
This project is fully responsive and works on mobile and desktop screens.

## Tech Stack
 **Frontend:** React.js, HTML5, CSS  
 **Design:** Figma (for UI/UX prototypes)  
 **Optional Workflow:** Locofy can convert Figma designs to code for faster development, though this project was implemented manually in React.

## Features
 Add new reminders with **title** and **description**  
 View all existing reminders in **personalized cards**  
 Soft pink aesthetic with **Playfair Display** font   
 Fully responsive design for mobile and desktop  
 Default example reminders included:
 Math assignment due tomorrow   
 Personal goal reminder   
 Other personalized reminders 

## How to Run
1. Clone the repository:  
   bash
   git clone <https://github.com/moria18/personal-reminder-board>
2. Navigate into project folder
     cd <ict project>
3. Install dependencies
    npm install
4. Start the development server
    npm start
5. Open your browser and go to http://localhost:3000 to view the app
